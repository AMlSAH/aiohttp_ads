# REST API сервис объявлений на aiohttp

**aiohttp** (асинхронный веб-фреймворк Python)

## Возможности

- Создание, получение, обновление и удаление объявлений.
- Получение списка всех объявлений.
- Хранение данных в памяти (словарь Python).

## Требования

- **Python 3.8+**
- Установленный пакет `aiohttp`

## Установка

1. Клонируйте репозиторий (если нужно) или сохраните файл `app.py`.
2. Установите зависимости:
3. `pip install aiohttp`
